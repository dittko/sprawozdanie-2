O projekcie

Aplikacja FootballAPP została stworzona jako projekt zaliczeniowy na przedmiot Web 2.0. To system , który łączy frontend oparty na Angularze z backendowym REST API napisanym w Node.js.

Główne funkcjonalności:

  • Pełna obsługa operacji CRUD dla piłkarzy i zespołów (dodawanie, edycja, usuwanie, przeglądanie)
  • System uwierzytelniania użytkowników: rejestracja, logowanie, aktywacja konta, zmiana oraz reset hasła
  • Interfejs responsywny – dostosowany do urządzeń mobilnych i desktopowych
  • Integracja z własnym API, w tym część dostępna wyłącznie po zalogowaniu (z użyciem tokenu JWT)

Technologie:

Frontend: Angular, TypeScript, CSS
Backend: Node.js, MongoDB
Uwierzytelnianie: JWT

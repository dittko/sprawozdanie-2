O projekcie

Aplikacja Football Manager Pro powstała jako projekt zaliczeniowy na przedmiot Programowanie Aplikacji Internetowych. Jest to pełnoprawny system łączący nowoczesny frontend w Angularze z REST API napisanym w Node.js.

Główne funkcjonalności:

  • Pełny CRUD dla ulubionych piłkarzy (dodawanie, edycja, usuwanie)
  • System autentykacji użytkowników (rejestracja, logowanie, JWT)
  • Responsywny interfejs dostosowany do wszystkich urządzeń
  • Integracja z własnym API zgodnie z wymaganiami projektu

Technologie:

Frontend: Angular 17, TypeScript, CSS
Backend: Node.js, Express, MongoDB
Autentykacja: JWT, bcrypt
